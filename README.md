# Автоматизация окружения для Frontend разработки

## Установка

```
npm i
npm i -g jshint
```

## Запуск

**Простой сервер**

```
npm run server
```

**Валидация HTML**

```
npm run validate-html
```

**Валидация JS (JSHint)**

```
npm run jshint
```

**Запуск DEV режима (сервер + валидаторы)**

```
npm run watch
```
